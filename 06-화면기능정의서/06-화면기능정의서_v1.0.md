## 2. 화면 상세 정의

### SCR-001: 로그인

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-001 |
| 화면명 | 로그인 |
| URL | /login |
| 설명 | 사용자 인증을 위한 로그인 화면 |

#### UI 요소

| 요소 ID | 유형 | 레이블 | 필수 | 유효성 검사 | 비고 |
|---------|------|--------|------|-------------|------|
| inp-email | Input | 이메일 | Y | 이메일 형식 | placeholder: "이메일을 입력하세요" |
| inp-password | Input | 비밀번호 | Y | 최소 8자 | type: password |
| btn-login | Button | 로그인 | - | - | Primary |

#### 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-001-01 | 로그인 | btn-login 클릭 | 이메일/비밀번호 검증 후 API 호출, JWT 토큰 저장 | POST /auth/login |
| FN-001-02 | 유효성 검사 | 입력 시 | 실시간 입력값 검증 | - |
| FN-001-03 | 계정 잠금 | 5회 실패 | 30분간 로그인 차단 | - |

#### 상태 처리

| 상태 | 표시 내용 | 조건 |
|------|-----------|------|
| 로딩 | 스피너 표시 | API 호출 중 |
| 성공 | 대시보드 이동 | 로그인 성공 |
| 실패 | 에러 메시지 | 인증 실패 |
| 잠금 | 잠금 안내 | 5회 연속 실패 |

### SCR-002: 메인 대시보드

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-002 |
| 화면명 | 메인 대시보드 |
| URL | /dashboard |
| 설명 | 역할별 맞춤 가젯으로 구성된 프로젝트 현황 대시보드 |

#### UI 요소 (역할별 가젯)

| 요소 ID | 유형 | 레이블 | 대상 역할 | 비고 |
|---------|------|--------|-----------|------|
| gadget-my-issues | Filter Result | 내 담당 이슈 | Developer | 오늘 할 일 |
| gadget-burndown | Chart | Sprint Burndown | Developer, PM | 스프린트 잔여량 |
| gadget-velocity | Chart | Velocity Chart | PM, 팀 리더 | 팀 생산성 추이 |
| gadget-sprint-report | Table | Sprint Report | PM, 팀 리더 | 완료/미완료 분석 |
| gadget-roadmap | Timeline | Epic 로드맵 | PM, PO | 전체 일정 |
| gadget-pie-type | Pie Chart | 이슈 타입별 분포 | PM, PO | 작업 구성 비율 |
| gadget-qa-queue | Filter Result | QA 대기 목록 | QA | 테스트 대기 이슈 |
| gadget-created-resolved | Chart | Created vs Resolved | QA | 버그 투입/해결 추이 |
| gadget-cfd | Chart | 누적 흐름 다이어그램 | 전체 | Cumulative Flow |

#### 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-002-01 | 가젯 로딩 | 페이지 진입 | 역할 기반 가젯 자동 구성 | GET /dashboard/gadgets |
| FN-002-02 | 이슈 클릭 | 이슈 항목 클릭 | 이슈 상세 화면 이동 | - |
| FN-002-03 | 가젯 커스터마이징 | 설정 버튼 | 가젯 추가/제거/재배치 | PUT /dashboard/layout |

### SCR-004: 스크럼 보드

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-004 |
| 화면명 | 스크럼 보드 |
| URL | /project/{key}/board |
| 설명 | 스프린트 단위 이슈를 상태별 컬럼으로 시각화하는 칸반 스타일 보드 |

#### UI 요소

| 요소 ID | 유형 | 레이블 | 비고 |
|---------|------|--------|------|
| col-backlog | Column | Backlog | 회색 배경 |
| col-selected | Column | Selected for Sprint | 파란색 배경 |
| col-progress | Column | In Progress | 노란색 배경, WIP 제한 표시 |
| col-review | Column | Code Review | 보라색 배경 |
| col-qa | Column | QA | 분홍색 배경 |
| col-done | Column | Done | 초록색 배경 |
| card-issue | Card | 이슈 카드 | 타입 아이콘, 우선순위, 담당자, SP 표시 |
| wip-indicator | Badge | WIP 제한 표시 | WIP 초과 시 빨간색 경고 |

| 기능 ID | 기능명 | 트리거 | 동작 | API |

| FN-004-01 | 드래그 앤 드롭 | 카드 드래그 | 이슈 상태 전환 (워크플로우 규칙 적용) | POST /transitions |
| FN-004-02 | WIP 경고 | 컬럼 이슈 초과 | 컬럼 헤더 색상 변경, 경고 표시 | - |
| FN-004-03 | 이슈 상세 | 카드 클릭 | 이슈 상세 모달/페이지 | GET /issue/{key} |
| FN-004-04 | 필터링 | 필터 선택 | 담당자/타입/우선순위별 필터링 | - |

### SCR-006: 이슈 상세

| 항목 | 내용 |
|------|------|
| 화면 ID | SCR-006 |
| 화면명 | 이슈 상세 |
| URL | /issue/{issueKey} |
| 설명 | 이슈의 전체 정보, 댓글, 첨부파일, 링크, 변경 이력을 표시 |

#### UI 요소

| 요소 ID | 유형 | 레이블 | 비고 |
|---------|------|--------|------|
| lbl-key | Label | 이슈 키 | PROJ-142 |
| sel-type | Select | 이슈 타입 | Epic/Story/Task/Bug/Sub-task |
| sel-status | Select | 상태 | 워크플로우 전환 규칙 적용 |
| sel-priority | Select | 우선순위 | Highest~Lowest |
| sel-assignee | Select | 담당자 | 팀원 목록 |
| inp-sp | Input | 스토리 포인트 | 피보나치 수열 |
| txt-description | TextArea | 설명 | Markdown 지원 |
| lst-comments | List | 댓글 목록 | @멘션 지원 |
| lst-links | List | 이슈 링크 | Blocks/Duplicates/Relates to |
| lst-history | List | 변경 이력 | Audit Log (필드별 변경 추적) |
| lst-attachments | List | 첨부파일 | 파일 업로드/다운로드 |

#### 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-006-01 | 필드 수정 | 필드 값 변경 | 인라인 편집, 자동 저장 | PUT /issue/{key} |
| FN-006-02 | 상태 전환 | 상태 변경 | 워크플로우 전환 규칙 검증 | POST /transitions |
| FN-006-03 | 댓글 작성 | 댓글 입력 | @멘션 시 알림 발송 | POST /issue/{key}/comment |
| FN-006-04 | 이슈 링크 | 링크 추가 | 관계 타입 선택 후 이슈 연결 | POST /issueLink |
| FN-006-05 | 워치 | Watch 토글 | 이슈 변경 알림 구독/해제 | POST /issue/{key}/watchers |
