## 0. 공통 UI 요소

### 0.1 GNB (Global Navigation Bar)

모든 인증 화면(SCR-002 이후)에 적용되는 상단 글로벌 내비게이션 바.

| 요소 ID | 유형 | 레이블 | 비고 |
|---------|------|--------|------|
| gnb-logo | Image/Link | LearnFlow AI 로고 | 클릭 시 역할별 대시보드로 이동 |
| gnb-search | Input | 강의 검색 | 강의명/태그 전체 검색, 단축키 `/` |
| gnb-notifications | IconButton | 알림 | 미확인 알림 수 뱃지 표시 |
| gnb-user-menu | Avatar/Menu | 사용자 메뉴 | 프로필, 설정, 로그아웃 |
| gnb-ai-indicator | Badge | AI 상태 | FinOps Kill-switch 활성 시 빨간 아이콘 표시 |

#### GNB 기능 정의

| 기능 ID | 기능명 | 트리거 | 동작 | API |
|---------|--------|--------|------|-----|
| FN-GNB-01 | 알림 조회 | gnb-notifications 클릭 | 최근 알림 드롭다운 | GET /api/v1/notifications |
| FN-GNB-02 | 강의 검색 | gnb-search 입력 | 강의 목록 필터링 | GET /api/v1/courses?q={keyword} |
| FN-GNB-03 | 로그아웃 | gnb-user-menu → 로그아웃 | JWT 무효화 + 로그인 페이지 이동 | POST /api/v1/auth/logout |
| FN-GNB-04 | AI 상태 표시 | Kill-switch 활성 시 | 빨간 배지 표시 + 툴팁 | GET /api/v1/admin/finops/status |

#### 레이아웃

```
┌──────────────────────────────────┬───────────────────────┐
│  [영상 플레이어]           70%    │  [AI 튜터 채팅]  30%  │

│  │     ▶  강의 영상          │   │  │ 무엇이든         │  │
│  │     (HLS 스트리밍)        │   │  │    물어보세요!   │  │
│  └──────────────────────────┘   │  │                 │  │
│  ┌──────────────────────────┐   │  │ ─ 추천 질문 ─   │  │
│  │  AI 요약 / 플래시카드     │   │  │ [N+1 문제란?]   │  │
│  └──────────────────────────┘   │  │ [Fetch 전략?]   │  │
│  ── 커리큘럼 ──                  │  │                 │  │
│  ✅ 1. 소개  ✅ 2. 기본         │  │ 👤 JPA에서      │  │
│  ▶️ 3. 심화  🔒 4. 실습        │  │  N+1 문제란?    │  │
```
